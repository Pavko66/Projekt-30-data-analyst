\# Projekt 30 – Data Analyst



To repozytorium dokumentuje moją drogę od podstaw do pracy jako Junior Data Analyst.



\## Cel



Celem projektu jest zdobycie umiejętności potrzebnych do pracy z danymi:

\- Excel

\- SQL

\- Power BI

\- Python

\- analiza danych

\- portfolio projektów



\## Struktura repozytorium



\- `01\_excel` – ćwiczenia i projekty w Excelu

\- `02\_sql` – zapytania SQL i ćwiczenia

\- `03\_power\_bi` – raporty i dashboardy

\- `04\_python` – analiza danych w Pythonie

\- `05\_portfolio\_projects` – gotowe projekty do portfolio

\- `06\_notes` – notatki z nauki

\- `datasets` – dane treningowe
