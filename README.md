# Projekt 30 – Data Analyst

To repozytorium dokumentuje moją drogę od podstaw do pracy jako Junior Data Analyst.

## Cel

Celem projektu jest zdobycie umiejętności potrzebnych do pracy z danymi:

- Excel
- SQL
- Power BI
- Python
- analiza danych
- portfolio projektów

## Struktura repozytorium

- `01_excel` – ćwiczenia i projekty w Excelu
- `02_sql` – zapytania SQL i ćwiczenia
- `03_power_bi` – raporty i dashboardy
- `04_python` – analiza danych w Pythonie
- `05_portfolio_projects` – gotowe projekty do portfolio
- `06_notes` – notatki z nauki
- `datasets` – dane treningowe
